About

This is a simple Flappy Bird–style game (NW-Bird-Game) implemented to demonstrate using the Amazon Q CLI tooling on Windows (with optional WSL). The project was built as part of a blog/tutorial and follows the resources linked below.

Features
- Classic side-scrolling bird game mechanics (flap to gain altitude, avoid obstacles)
- Built with the Amazon Q CLI development workflow
- Windows-friendly development notes (WSL guidance included)

Prerequisites
- Windows 10/11 (or WSL2 on Windows)
- Amazon Q CLI: installation instructions at https://docs.aws.amazon.com/amazonq/latest/qdeveloper-ug/command-line-installing.html
- (Optional) Windows Subsystem for Linux (WSL): https://learn.microsoft.com/en-us/windows/wsl/setup/environment

Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/sunilkumarald/NW-Bird-Game.git
   cd NW-Bird-Game
   ```
2. Follow the Amazon Q CLI install instructions linked in the Prerequisites.
3. If using WSL, ensure WSL is configured and you have access to required tooling.

Running the game
- Follow the steps in the repository to build and run the game using the Amazon Q CLI. See project scripts or the blog post for a step-by-step walkthrough.

Development
- Use the Amazon Q CLI to develop and test locally. Refer to the official docs and the blog post for examples and walkthroughs.
- Make sure tooling (node/python/other runtime) required by the project is installed per any project scripts.
